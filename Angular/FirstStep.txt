Create a Folder "MyFolder"
> CD MyFolder
Install latest angular cli in the folder locally:
> npm install --save-dev @angular/cli@latest
> CD..
Create a new angular app:
> ng new MyFolder
Use SCSS syntax. It's newer than Sass syntax!
> CD MyFolder
Add angular material:
> ng add @angular/material

Check node/npm version:
node -v
npm -v
Check angular version:
ng version
First time install angular-cli globally:
npm install -g @angular/cli
Upgrade gloablly installed angular-cli to latest version:
npm install -g @angular/cli@latest
